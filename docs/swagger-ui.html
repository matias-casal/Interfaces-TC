The real-time service uses Socket.io for WebSocket communication. Connect to http://localhost with your JWT token.

Connection Example:

const socket = io('http://localhost', {
  auth: {
    token: 'your-jwt-token'
  },
  transports: ['websocket', 'polling']
});

new_message INCOMING

Receive new message in real-time when someone sends you a message.

socket.on('new_message', (message) => {
  console.log('New message:', message);
  // message: { id, senderId, receiverId, encryptedText, status, timestamp }
});

message_status INCOMING

Receive updates when your sent messages are delivered or read.

socket.on('message_status', (status) => {
  console.log('Status update:', status);
  // status: { messageId, status: 'delivered' | 'read' }
});

user_typing INCOMING

Receive typing indicators from other users.

socket.on('user_typing', (data) => {
  console.log('Typing:', data);
  // data: { userId, typing: true | false }
});

mark_delivered OUTGOING

Mark messages as delivered when you receive them.

socket.emit('mark_delivered', ['messageId1', 'messageId2']);

mark_read OUTGOING

Mark messages as read when user views them.

socket.emit('mark_read', ['messageId1', 'messageId2']);

typing_start / typing_stop OUTGOING

Send typing indicators to other users.

// Start typing
socket.emit('typing_start', { receiverId: 'user-123' });

// Stop typing
socket.emit('typing_stop', { receiverId: 'user-123' });
